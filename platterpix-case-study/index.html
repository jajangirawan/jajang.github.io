Story

Client asked me to redesign a mobile application. This app is US only. He is interested to have modern and minimalist design without deducting User Experience.

Mobile application for finding nearby restaurant and foods. This Application helps users to find restaurant, table reservation, and food ordering.

Goal Project

This application was made to help users finding food or restaurant nearby. User is able to book restaurant and order menu from this app.
User is able to send review for restaurant or food, see his/her history (eat journal), look for discountm etc.
This application is simple with a lot of features.
